# README

Code repository for *Methods to Account for Unobserved Baseline Confounders with Time-Invariant and Time-Varying Effects in Cross-Lagged Panel Research*.

The main directory contains two sub-directories:
- `Population Analyses` contains files to run the population analyses.
- `Simulation Study` contains the files to run the simulation study.

Both directories contain their own respective README files with more information on their contents and directions to run the analyses. Alternatively, to run both the population analyses and simulation study in one go, open a terminal in the current directory, and run `bash run_sim_pop.sh`.
